# CodeAlpha Internship Tasks

## Task 1: Language Translator

### Description
Language Translator is a Python-based application developed as part of the CodeAlpha Internship Program. It allows users to translate text from one language to another.

### Features
- Translate text between multiple languages
- Simple and user-friendly interface
- Fast and accurate translations
- Supports various languages

### Technologies Used
- Python
- Streamlit
- Google Translate API / Translator Library

## Task 2: FAQ Chatbot

### Description
FAQ Chatbot is a Python-based application that answers frequently asked questions using predefined questions and answers. It provides quick and accurate responses based on user input.

### Features
- Answers common user questions
- User-friendly interface
- Uses text similarity for matching questions
- Fast response generation

### Technologies Used
- Python
- Streamlit
- Scikit-learn
